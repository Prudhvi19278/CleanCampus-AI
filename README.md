# CleanCampus AI

CleanCampus AI is a conceptual AI-based campus cleanliness monitoring project
developed for the 1M1B AI for Sustainability Virtual Internship.

## Overview
The project proposes an AI-driven approach to identify, categorize, and manage
cleanliness issues on university campuses.

## SDG Alignment
- SDG 11: Sustainable Cities and Communities
- SDG 12: Responsible Consumption and Production

## Note
This is a prototype-level project.
No real AI model or software implementation is deployed.
